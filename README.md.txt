# AKS Helm Lab

Learning project for deploying applications to Azure Kubernetes Service using Helm charts.

## Project Structure
- `/terraform` - Infrastructure as Code for AKS cluster and supporting resources
- `/helm-charts` - Custom Helm charts for applications
- `/docs` - Documentation and diagrams

## Phases
- Phase 1: Azure Infrastructure Setup (AKS, ACR, Networking)
- Phase 2: Application Development and Containerization
- Phase 3: Helm Chart Development
- Phase 4: Deployment and Configuration
- Phase 5: Advanced Helm Features
